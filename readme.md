# general webpack

> 自己动手写一个自己项目中要用的webpack配置

## Usage

下载项目
```
    git clone https://github.com/zengshide123/general-webpack.git
```
安装依赖
```
    cnpm i 
```
开发项目
```
    npm start
```
打包项目
```
   npm run build
```
## 说明

- 为单入口文件
- 支持less,scss
- 支持postcss
- 支持图片转base64
- 支持es6转es5
- 支持热更新
- 请注意暂时不兼容IE8
